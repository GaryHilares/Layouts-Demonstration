Fixed layout

What is a fixed layout?

A fixed layout is the layout whose size is fixed regardless of the size of the viewport and other variables.

How to create it with CSS?

Write the widths of your elements using pixels in CSS.

Advantages of a fixed layout

  • Sizes are predictable.
  • Easier to make than fluid and elastic layouts.

Disadvantages of a fixed layout
