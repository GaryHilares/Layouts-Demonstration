Elastic layout

What is an elastic layout?

An elastic layout resizes itself depending on the font size which the client uses.

How to create it with CSS?

Write the widths of your elements using ems or rems in CSS.

Advantages of an elastic layout

  • Sizes are somewhat predictable.
  • The user gets control of the size of the layout.

Disadvantages of an elastic layout

  • There may still be extra whitespace to the sides.
  • Usually not as responsive as the fluid layout.
